# PHP fpm

The stock PHP fpm image with some added tools for use in local development.

# Versions

Currently images are being built for:
* 7.3
* 7.3-blckfire
* 7.3-xdebug
* 7.2
* 7.2-blckfire
* 7.2-xdebug

# Added PHP modules
* blackfire
* bz2
* gd
* iconv
* intl
* mysqli
* opcache
* pdo_mysql
* redis
* soap
* xdebug
* zip
